# Instalación de pm2

1. Instalar pm2

`npm install -g pm2`

1. Comandos
- Correr una aplicación
    
    `pm2 start app.js`
    
- Detener una aplicación
    
    `pm2 stop app.js`
    
- Reiniciar una aplicación
    
    `pm2 restart app.js`
    
- Listar todas las aplicaciones
    
    `pm2 list`
    
- Eliminar una aplicación de la lista
    
    `pm2 delete app.js`
