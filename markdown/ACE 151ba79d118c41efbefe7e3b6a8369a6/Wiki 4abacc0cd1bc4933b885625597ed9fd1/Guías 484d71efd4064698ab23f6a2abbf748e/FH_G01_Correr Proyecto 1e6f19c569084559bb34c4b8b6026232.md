Es necesario tener corriendo el backend y el frontend simultáneamente.

## Despúes de la primera vez

Para correr front:

```markdown
npm start
```

Para detener front:

```markdown
ctrl + c
```

Para correr back:

```markdown
pm2 start serverDEV.json
```

Para ver los logs del servidor: (Abrirlo en una terminal mientras corre)

```markdown
pm2 logs 
```
